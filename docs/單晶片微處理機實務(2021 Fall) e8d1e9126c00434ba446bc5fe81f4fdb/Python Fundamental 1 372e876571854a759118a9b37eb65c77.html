課程的 Python 作業樣板

Template 1 : 不需要輸入資料，處理後直接輸出結果

#! /bin/python3

def main():
	print("Hello, Python world!")

if __name__=="__main__":
    main()
    exit(0)
#! /bin/python3

def drawTriangle(l):
    seq = range(1,l+1)
    if (l<0):
        seq = range(-l,0,-1)
    for i in seq:
        print("*"*i)
        
def main():
    drawTriangle(5)
    print()
    drawTriangle(-5)
    
if __name__=="__main__":
    main()
    exit(0)

Template 2 : 由終端機輸入「整數/浮點數/字串」資料，並在處理後輸出結果

#! /bin/python3

def doSomething(a,b):
    print(a+b)

def main():
    a = int(input())
    b = int(input())
    doSomething(a,b)
 
if __name__=="__main__":
    main()
    exit(0)
#! /bin/python3

def sum(nums):
    s = 0
    for n in nums:
        s+=n
    return s

def main():
    nums = [int(v) for v in input().split()]
    print(sum(nums))
 
if __name__=="__main__":
    main()
    exit(0)

課堂練習

  • 請參考作業樣板程式 Template 1，印出一個8層的正三角形。
def drawRTriangle(l):
    for i in range(1,l+1):
        print(" "*(l-i)+"*"*(i*2-1))
        
def main():
    drawRTriangle(8)
    
if __name__=="__main__":
    main()
    exit(0)
  • 請參考作業樣板程式 Template 2，由終端機輸入任意「浮點數」數列，經比對判斷後輸出數列中的最大值。 
#! /bin/python3

def maxF(nums):
    m = nums[0]
    for f in nums[1:]:
        if f>m:
            m = f
    return m

def main():
    nums = [float(v) for v in input().split()]
    print(maxF(nums))
 
if __name__=="__main__":
    main()
    exit(0)
